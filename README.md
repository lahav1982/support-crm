# SupportAI – Email Support Agent

An AI-powered customer support inbox built with React + Claude AI.

## Folder Structure
```
email-support-agent/
├── index.html         ← entry point
├── package.json       ← project config
├── vite.config.js     ← build config
└── src/
    ├── main.jsx       ← React bootstrap
    └── App.jsx        ← main app (edit this)
```

## Deploying to Vercel
1. Upload this entire folder to your GitHub repo
2. Connect the repo in Vercel
3. Vercel will auto-detect it as a Vite project — no settings needed
4. Click Deploy ✓
